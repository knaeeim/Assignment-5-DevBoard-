Question-1: What are the different ways to select an element in the DOM?

Answer: There are five ways to select an element in the DOM. If I give a Example How can we select a element in five differnt ways.

  • document.getElementById("specific Id Name")
    • Explaination : getElementById is select element by it's id name
    • Example : document.getElementById("myId")
  • document.getElementsByClassName("specific Class Name")
    • Explaination : getElementsByClassName is select element by it's class name
    • Example : document.getElementsByClassName("myClass")
  • document.getElementsByTagName("specific Tag Name")
    • Explaination : getElementsByTagName is select element by it's tag name
    • Example : document.getElementsByTagName("div")
  • document.querySelector("CSS Selector")
    • Explaination : querySelector is select element by it's css selector
    • Example : document.querySelector(".myClass")
  • document.querySelectorAll("CSS Selector")
    • Explaination : querySelectorAll is select all element by it's css selector
    • Example : document.querySelectorAll(".myClass")

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer: innerHTML, innerText, and textContent are the properties of the DOM. These properties are used to get or set the content of an element.

innerHTML:

innerHTML is used to return text with HTML Structure. You can also set the HTML structure by using this element from the JavaScript. If you set the innerHTML property, it will replace the content of the element with the new content.

innerText:

innerText is used to return the text of an element. It will return the text of the element without HTML structure. If you set the innerText property, it will replace the content of the element with the new content.

textContent:

textContent is used to return the text of an element with Hidden Text. it will replace the content of the element with the new content.

Question-3: What is event delegation in the DOM?

Answer: Event Delegation is a technique in which we attach an event listener to a parent element instead of attaching eventlisteners to the child elements. This technique is very useful when we have a large number of child elements to handle. It improves the performance of the application because it reduces the number of event listeners.

Question-4: What is event bubbling in the DOM?

Answer: Event Bubbling is a mechanism in which the event starts from the target element and bubbles up to the parent elements. When an event occurs on an element, it first runs the event handler on the target element, then on the parent element, and so on until it reaches the root element. This technique is used to handle the event on multiple elements.

Question-5: How do you create, add, and remove elements using JavaScript?

Answer: We can create, add, and remove elements using JavaScript by using the following methods:

Create Element:

We can create an element using the document.createElement() method. This method creates a new element with the specified tag name.

Add Element:

We can add an element to the DOM using the appendChild() method. This method appends the specified element as the last child of the parent element.

Remove Element:

We can remove an element from the DOM using the removeChild() and remove() method. This method removes the specified element from the parent element.
